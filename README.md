# GlyphForge

Warm editorial SVG typography studio built with Flask.

GlyphForge is a cozy creative web app that generates atmospheric SVG typography with niche editorial aesthetics, soft gradients, and experimental visual styles.

---

## Features

- SVG typography generation
- Live preview rendering
- User authentication
- Responsive glassmorphism layout
- Flask backend
- SVG export system
- Multiple typography styles

---

## Tech Stack

- Python
- Flask
- HTML5
- CSS3
- JavaScript

## Run Locally

Clone repository:

```bash
git clone https://github.com/eshawagh/GlyphForge.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run Flask app:

```bash
python app.py
```

Open browser:

```bash
http://127.0.0.1:5000
```
